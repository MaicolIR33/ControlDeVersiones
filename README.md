Este proyecto consiste en la implementación de dos funciones desarrolladas en JavaScript con el objetivo de practicar conceptos fundamentales del lenguaje. La primera función, suma(a, b), recibe dos valores numéricos como parámetros y retorna el resultado de su suma, permitiendo realizar operaciones matemáticas simples mediante el uso de funciones y parámetros. La segunda función, validarEmail(email), tiene como propósito verificar si un texto cumple con una estructura básica de correo electrónico; para ello, valida que el texto contenga el símbolo @, que incluya al menos un punto ., que el @ no se encuentre al inicio y que el último punto esté ubicado después del @, asegurando así una forma mínima correcta como "usuario@dominio.com
". Ambas funciones aplican estructuras condicionales y métodos de manipulación de cadenas como includes, indexOf y lastIndexOf, reforzando el uso de validaciones lógicas y buenas prácticas en la organización del código, con un enfoque académico orientado al aprendizaje y fortalecimiento de la lógica de programación.
